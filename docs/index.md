# 关于我们

## 关于公司

我们是一家深耕对日软件研发的企业，立足中国，依托离岸开发优势，拥有充足的技术人才和具竞争力的人力成本。我们支持日元结算，为客户提供顺畅便捷的合作体验。

我们坚持“更快、更好”的交付原则，为每位客户配置专属团队，确保专注度与执行效率。面对变化，我们保持开放姿态，拥抱迭代，持续调整，以最合适的方式回应客户与市场的需求，交付可靠的解决方案与服务体验。

## 关于团队

我们的团队由一群保持高速增长势头的年轻工程师组成，执行力强、反馈快。大家始终保持自省与开放心态，营造共学、共创的氛围。团队成员以编程为核心追求，以近乎完美的标准要求交付质量。

目前团队已进入发展过程中的规范阶段，协作更顺畅、沟通更高效、流程更清晰，每个人都理解自己的职责与边界。这种稳健高效的工作状态，帮助我们在复杂项目中依旧保持高质量输出。

## 加入我们

我们坚持持续创新与自我进化，相信唯有主动变革，才能在激烈竞争中保持优势。自由与开放是我们的文化底色，多元化思维与创新精神在这里得到充分鼓励。

**我们寻找的人才**  
希望与我们一起成长、拥抱变化的你。如果你期待在充满活力与机会的环境中挑战自我、不断求新，我们诚邀加入。

**职位要求**

- 快速适应不同场景与需求，敢于面对不确定性
- 具备创新意识，乐于提出新想法并验证多种方案
- 善于跨团队协作，愿意与不同背景的伙伴携手前进
- 拥有自驱力与进取心，能够主动发现问题并推动解决

**加入我们的理由**  
我们倡导开放、包容的氛围，尊重每一个观点。多样性是我们的重要资产，每位成员都能在这里获得充分信任与支持。面向未来，我们希望和你一起成为行业的引领者。

**如何申请**
