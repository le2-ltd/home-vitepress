# About Us


## 会社概要
私たちは日本向けのソフトウェア開発に特化した会社であり、中国国内でソフトウェアを開発しています。オフショアの利点を活かし、より豊富な技術者リソースと低コストの人件費を実現しています。また、円決済を採用しており、クライアントにさらなる便利さを提供しています。

私たちはクライアントに対して「より速く、より良くプロジェクトを完了する」という原則を持っています。そのため、特定の担当者が一つのプロジェクトのみを担当し、集中力と効率を確保しています。さらに、変化を受け入れ、柔軟に調整することで、絶えず変化する市場とクライアントのニーズに対応しています。この方法により、私たちはクライアントに最適なソリューションとサービス体験を提供できると信じています。

## チーム概要
私たちのチームは活力に満ちた若者たちで構成されており、迅速に対応する能力を持っています。常に反省とオープンな態度を維持し、積極的な学習雰囲気を醸成しています。チームメンバーはプログラミングに専念し、完璧主義を目指しており、各プロジェクトを高品質で完了させることに努めています。

現在、チームは発展段階の標準化期にあります。メンバー間の協力がより円滑になり、コミュニケーションが効率的で、作業フローが明確になっています。お互いの役割と責任をよりよく理解することで、作業効率とプロジェクトの品質を保証しています。これにより、複雑なプロジェクトに直面しても、高効率な出力と高品質な成果を維持することができます。

## 採用情報
私たちは継続的なイノベーションと発展に力を入れている企業です。絶えず変革し適応することのみが、激しい市場競争で勝ち残る道だと深く理解しています。当社の企業文化は自由とオープンを核心としており、多様な思考と創造精神を奨励しています。

**求める人材**
私たちは、共に成長し時代の変遷に適応できる優秀な人材を探しています。活力と機会に満ちた環境で成長し、挑戦に立ち向かい、イノベーションを開拓する意欲があるなら、ぜひ私たちに加わってください。

迅速に新しい環境に適応し、変化に対応できる強い適応能力が必要です。また、イノベーション精神を重視し、新しいアイデアを提案し、異なる解決策を試すことを楽しめることが重要です。チームワークに優れ、異なる背景や視点を持つ同僚と協力して共通の目標を達成できることを期待しています。さらに、強い向上心と自己駆動力を持ち、不確実な未来の中でチャンスを見つけることが求められます。
