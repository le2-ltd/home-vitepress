## 会社概要
当社は日本向けソフトウェア開発企業であり、中国国内でのソフトウェア開発に特化しています。オフショアの利点を活かし、豊富な技術人材リソースと低い人件費を有しています。また、日本円での決済を行い、お客様により多くの便益を提供しています。
当社の顧客対応の原則は、プロジェクトをより速く、より良く完了させることにあります。そのため、1つのプロジェクトに専任のスタッフを配置し、専心して効率的に作業を行っています。さらに、変更を受け入れ、柔軟に調整し、常に市場や顧客の要求に対応しています。このようなアプローチにより、お客様に最適なソリューションとサービス体験を提供できると信じています。
